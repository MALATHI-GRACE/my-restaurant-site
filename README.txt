# Restaurant Website Project

This is a front-end project for a restaurant website built using HTML, CSS, JavaScript, and Bootstrap.

## Features
- **Home Page**: Overview of the restaurant (using Bootstrap for layout and styling).
- **Menu Page**: Detailed list of available dishes (using JavaScript for dynamic content).
- **Order Online**: Contains the process related to online orders.
- **Services**:
  - **Catering Service**: Form for catering inquiries.
  - **Event Service**: Form for event inquiries.
- **About**: The story of the restaurant.
- **Contact**: Contact details and form.

## Technologies Used
- HTML
- CSS
- JavaScript (Menu Page)
- Bootstrap (Home Page)

## Future Enhancements
- Add a back-end with a database to store form submissions.
